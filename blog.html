Question 1: Difference between localStorage and sessionStorage?

Answer: localStorage - stores data with no expiration date. window. sessionStorage - stores data for one session. The difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends.

Question 2: Difference between global scope and block scope?

Answer: Global Scope is a variables declared Globally (outside any function) have Global Scope. Global variables can be accessed from anywhere in a JavaScript program. Block Scope is a variable when declared inside the if or switch conditions or inside for or while loops, are accessible within that particular condition or loop.

Question 4: How will you get undefined value in javascript?

Answer: A variable or an object has an undefined value when no value is assigned before using it.
You will get undefined value when you call a non-existent property or method of an object.
If you pass less arguments in function call then, that parameter will have undefined value.
